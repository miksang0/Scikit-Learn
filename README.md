# Scikit-Learn

This repository is dedicated to learning and implementing machine learning concepts using **Scikit-Learn**.

## Topics Covered

The repository is structured to cover the following key topics:

1. **Datasets**
2. **Splitting Data**
3. **Preprocessing**
4. **Feature Encoding**
5. **Classification**
6. **Regression**
7. **Clustering**
8. **Principal Component Analysis (PCA)**
9. **Evaluation Metrics**
10. **Cross-Validation**
11. **Hyperparameter Tuning**
12. **Pipelines**

## Purpose

The goal of this repository is to provide a structured, hands-on approach to understanding machine learning workflows using Scikit-Learn — from data preparation to model evaluation and optimization.
